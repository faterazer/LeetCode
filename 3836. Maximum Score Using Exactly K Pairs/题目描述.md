# 3836. 恰好 K 个下标对的最大得分

给你两个长度分别为 `n` 和 `m` 的整数数组 `nums1` 和 `nums2`，以及一个整数 `k`。

你必须 **恰好** 选择 `k` 对下标 $(i_1, j_1), (i_2, j_2), ..., (i_k, j_k)$，使得：

- $0 \leq i_1 < i_2 < \dots < i_k < n$
- $0 \leq j_1 < j_2 < \cdots < j_k < m$

对于每对选择的下标 `(i, j)`，你将获得 `nums1[i] * nums2[j]` 的得分。

总 **得分** 是所有选定下标对的乘积的 **总和**。

返回一个整数，表示可以获得的 **最大** 总得分。

**示例 1：**

```text
输入： nums1 = [1,3,2], nums2 = [4,5,1], k = 2

输出： 22

解释：

一种最优的下标对选择方案是：

- (i_1, j_1) = (1, 0)，得分为 3 * 4 = 12
- (i_2, j_2) = (2, 1)，得分为 2 * 5 = 10

总得分为 12 + 10 = 22。
```

**示例 2：**

```text
输入： nums1 = [-2,0,5], nums2 = [-3,4,-1,2], k = 2

输出： 26

解释：

一种最优的下标对选择方案是：

- (i_1, j_1) = (0, 0)，得分为 -2 * -3 = 6
- (i_2, j_2) = (2, 1)，得分为 5 * 4 = 20

总得分为 6 + 20 = 26。
```

**示例 3：**

```text
输入： nums1 = [-3,-2], nums2 = [1,2], k = 2

输出： -7

解释：

最优的下标对选择方案是：

- (i_1, j_1) = (0, 0)，得分为 -3 * 1 = -3
- (i_2, j_2) = (1, 1)，得分为 -2 * 2 = -4

总得分为 -3 + (-4) = -7。
```

**提示：**

- `1 <= n == nums1.length <= 100`
- `1 <= m == nums2.length <= 100`
- `-10^6 <= nums1[i], nums2[i] <= 10^6`
- `1 <= k <= min(n, m)`
