# 412. Fizz Buzz

写一个程序，输出从 *1* 到 *n* 数字的字符串表示。

1. 如果 *n* 是 3 的倍数，输出 "Fizz"；
2. 如果 *n* 是 5 的倍数，输出 "Buzz"；
3. 如果 *n* 同时是 3 和 5 的倍数，输出 "FizzBuzz"。

**示例：**

```()
n = 15,

返回:
[
    "1",
    "2",
    "Fizz",
    "4",
    "Buzz",
    "Fizz",
    "7",
    "8",
    "Fizz",
    "Buzz",
    "11",
    "Fizz",
    "13",
    "14",
    "FizzBuzz"
]
```
