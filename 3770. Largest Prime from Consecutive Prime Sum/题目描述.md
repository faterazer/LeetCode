# 3770. 可表示为连续质数和的最大质数

给你一个整数 `n`。

返回小于或等于 `n` 的**最大质数**，该质数可以表示为从 2 开始的一个或多个 **连续质数** 之和。如果不存在这样的质数，则返回 0。

质数是大于 1 的自然数，且只有两个因数：1 和它本身。

**示例 1：**

```text
输入： n = 20

输出： 17

解释：

小于或等于 n = 20，且是连续质数和的质数有：

 - 2 = 2

  - 5 = 2 + 3

  - 17 = 2 + 3 + 5 + 7

其中最大的质数是 17，因此答案是 17。
```

**示例 2：**

```text
输入： n = 2

输出： 2

解释：

唯一小于或等于 2 的连续质数和是 2 本身。
```

**提示：**

- `1 <= n <= 5 * 10^5`
