# 253. 会议室 II

给你一个会议时间安排的数组 `intervals` ，每个会议时间都会包括开始和结束的时间 `intervals[i] = [start-i, end-i]` ，返回 *所需会议室的最小数量* 。

**示例 1：**

```()
输入：intervals = [[0,30],[5,10],[15,20]]
输出：2
```

**示例 2：**

```()
输入：intervals = [[7,10],[2,4]]
输出：1
```

**提示：**

- `1 <= intervals.length <= 10^4`
- `0 <= start-i < end-i <= 10^6`
