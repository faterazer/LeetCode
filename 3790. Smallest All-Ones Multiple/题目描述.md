# 3790. 最小全 1 倍数

给你一个正整数 `k`。

找出满足以下条件的 **最小** 整数 `n`：`n` 能被 `k` 整除，且其十进制表示中 **只包含数字 1**（例如：1、11、111、……）。

返回一个整数，表示 `n` 的十进制表示的 **位数** 。如果不存在这样的 `n`，则返回 `-1`。

**示例 1：**

```text
输入： k = 3

输出： 3

解释：

n = 111，因为 111 能被 3 整除，但 1 和 11 不能。n = 111 的长度为 3。
```

**示例 2：**

```text
输入： k = 7

输出： 6

解释：

n = 111111。n = 111111 的长度为 6。
```

**示例 3：**

```text
输入： k = 2

输出： -1

解释：

不存在满足条件且为 2 的倍数的有效 n。
```

**提示：**

- `2 <= k <= 10^5`
